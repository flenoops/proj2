CSS — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.

Проще говоря, мы описываем браузеру как должны выглядеть элементы сайта. Итак, разберёмся с тем, как это работает. Как и с HTML, мы пишем это в специальном файле с расширением .css, который можно открыть и изменять в обычном блокноте. Однако рекомендуются специально предназначеные для этого программы - Notepad++ (универсальный редактор кода) или Sublime Text 3.

Как и с HTML, разберёмся с некоторыми вещами:

  • CSS - это не язык программирования или разметки, это язык описания.
  • CSS всегда используется совместно с HTML, сам по себе он бесполезен. Поэтому, сначала нужно сверстать хотя бы простейшую HTML-страницу.
  • Все описания CSS строятся по единому принцпу: 

					сепаратор{
					    атрибут1: значение1;
					    атрибут2: значение2;
					    атрибут3: значение3;
					       ...       ...
					    атрибутN: значениеN;
					}

  • Пример: 

					p{
					    font-family: verdana;
					    font-size: 1.5em;
					    color: blue;
					}

Данный код влияет на текст. Рассмотрим подробнее:

  • p - это сепаратор тега <p>. Далее в фигурных скобках мы указываем параметры, которые будут влиять на весь текст, который расположен между тегами <p> </p>.
